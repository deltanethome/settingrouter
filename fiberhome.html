Panduan Mengganti Nama & Password WiFi FiberHome D2

📶 FiberHome HG6145D2

Ikuti langkah berikut untuk mengganti nama WiFi (SSID) dan password WiFi pada modem FiberHome D2.

1Hubungkan ke WiFi

Pastikan HP atau laptop sudah terhubung ke WiFi modem FiberHome D2.

2Buka Halaman Modem

Buka browser lalu ketik:

http://192.168.1.1

3Login

Username : user
Password : user1234

4Masuk Menu WiFi

Pilih menu:

Network → WLAN → 2.4G Advanced

5Ganti Nama WiFi

Cari kolom SSID Name kemudian ubah sesuai keinginan.

Contoh:
JAFIN NETWORK

6Ganti Password WiFi

Cari kolom WPA Passphrase lalu isi password baru.

Contoh Password:
Jafin@2026

7Simpan Pengaturan

Klik tombol Apply atau Save.

Tunggu beberapa detik hingga modem menyimpan konfigurasi.

Butuh Bantuan?

Jika modem tidak bisa login atau internet tidak tersambung, silakan hubungi teknisi Jafin Network.
