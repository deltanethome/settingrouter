Urutan setting

Ganti nama WiFi dan password

Pastikan perangkat sudah tersambung ke WiFi modem sebelum membuka halaman login. Setelah pengaturan disimpan, perangkat perlu tersambung ulang memakai nama dan password baru.

1Hubungkan perangkat

Sambungkan HP atau laptop ke WiFi FiberHome yang ingin diubah.

2Buka halaman modem

Buka browser, lalu akses http://192.168.1.1. Jika tidak terbuka, coba cek gateway WiFi dari perangkat.

3Login admin

Masuk menggunakan akun yang diberikan teknisi. Beberapa unit memakai username user dan password user1234.

4Buka menu WLAN

Masuk ke menu Network, lalu pilih bagian WLAN untuk pengaturan WiFi.

5Ubah WiFi 2.4G

Pada bagian WLAN 2.4G, ubah nama SSID dan password sesuai kebutuhan pelanggan, lalu simpan.

6Ubah WiFi 5G

Jika modem mendukung dual band, ulangi pengaturan pada WLAN 5G.
