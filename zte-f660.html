Step 1: Hubungkan HP atau laptop ke WiFi modem ZTE F660.
Step 2: Buka browser lalu akses 192.168.1.1.
Step 3: Login menggunakan username dan password modem:
user / @admin123
Step 4: Masuk ke menu Network → WLAN → SSID Settings
Step 5: Pada kolom SSID Name ubah nama WiFi sesuai keinginan.
Step 6: Untuk mengganti password, buka menu Security / WPA Passphrase
Step 7: Masukkan password baru minimal 8 karakter kemudian klik Apply / Save.
Step 8: Sambungkan kembali perangkat menggunakan nama dan password baru.
